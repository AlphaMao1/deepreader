# Changelog

## v0.2.0（开发中）

基于 [SageRead](https://github.com/xincmm/sageread) 二次开发，在原有阅读器功能之上增加以下能力：

### 新增
- **AI 记忆系统**：三区记忆架构（用户偏好 / 书籍知识 / 概念定义），跨 session 持续积累，自动在后台提取关键信息
- **意图诊断**：AI 回答前先判断用户真实意图（验证理解 / 概念解释 / 追问原理 / 延伸联想），减少"答非所问"
- **智能工具路由**：L0 对话历史 → L1 当前页面 → L2 RAG → L3 网络搜索，能不调工具就不调
- **Header 常驻章节导航**：上/下章按钮常驻 Header 两侧，不再依赖 hover
- **划线开新对话**：文本高亮 → Ask AI 自动开新线程，不污染当前对话上下文
- **`/记住` 斜杠命令**：手动触发长期记忆提取和保存
- **记忆提取模型选择**：设置页可独立配置后台记忆提取使用的轻量模型
- **Token 优化**：技能注入改为仅注入名称+描述（首次需要时调 `getSkills` 拉完整流程），书籍元信息首轮后不重复注入

### 修复
- 笔记格式化：去掉二次确认步骤，直接输出并同步保存到 Obsidian

## v0.1.1（原 SageRead 基础版本）

> 以下为上游 [SageRead](https://github.com/xincmm/sageread) 的功能基线

- EPUB 格式支持，滚动 / 分页双模式
- AI 对话助手，RAG 书内语义检索
- 高亮标注、书签、文本摘录
- 阅读统计可视化
- 自定义 AI 技能（提示词模板）
- 全文搜索
- TTS 朗读
- 深色 / 浅色主题，自定义字体布局
- Obsidian 导出
- 思维导图生成
- 本地数据存储，支持自托管 AI
